## 아이디어 1: Swiss Brutalism Meets Luxury

**Design Movement**: 스위스 브루탈리즘 + 럭셔리 미니멀리즘의 하이브리드

**Core Principles**:
- 극단적인 타이포그래피 계층 구조 (초대형 헤드라인 vs 극소형 디테일)
- 비대칭적 그리드 시스템으로 긴장감 조성
- 네거티브 스페이스를 적극적 디자인 요소로 활용
- 절제된 색상과 대담한 구조의 대비

**Color Philosophy**: 
모노크롬 베이스 (크림 화이트 #FAFAF8, 차콜 블랙 #1A1A1A)에 단 하나의 액센트 컬러 (딥 포레스트 그린 #1B3A2F)만 사용. 색상은 감정이 아닌 정보 계층을 전달하는 도구로만 기능.

**Layout Paradigm**: 
비대칭 분할 레이아웃. 화면을 60:40 또는 70:30으로 분할하여 한쪽은 대형 비주얼, 다른 쪽은 타이포그래피 중심 정보. 스크롤 시 섹션이 슬라이드처럼 전환되는 풀스크린 경험.

**Signature Elements**:
- 초대형 세리프 숫자 (제품 번호를 200px+ 크기로 배경 요소화)
- 1px 두께의 정밀한 구분선이 공간을 기하학적으로 분할
- 호버 시 확대되는 제품 이미지 (1.05배 스케일, 600ms ease-out)

**Interaction Philosophy**: 
모든 인터랙션은 "정밀함"을 표현. 버튼은 없고 밑줄이 그어진 텍스트 링크만 존재. 호버 시 밑줄이 왼쪽에서 오른쪽으로 그려지는 애니메이션 (300ms).

**Animation**: 
페이지 로드 시 요소들이 순차적으로 페이드인 (각 100ms 딜레이). 스크롤 시 패럴랙스 효과로 배경 이미지는 느리게, 텍스트는 빠르게 이동. 모든 트랜지션은 cubic-bezier(0.4, 0, 0.2, 1) 사용.

**Typography System**: 
- Display: Playfair Display (900 weight) - 제품명, 대형 숫자
- Body: Inter (400, 500) - 설명, 디테일
- Accent: Space Mono (400) - 제품 코드, 기술 스펙
크기 비율: 72px (h1) / 48px (h2) / 16px (body) / 12px (caption)

## 아이디어 2: Japanese Zen Minimalism

**Design Movement**: 일본 선(禪) 미학과 와비사비 철학

**Core Principles**:
- 완벽한 불완전함 (의도적 비대칭, 자연스러운 여백)
- 물질성과 촉각적 질감 강조
- 고요함과 집중을 유도하는 공간 설계
- 자연 요소와 디지털의 조화

**Color Philosophy**: 
자연에서 추출한 미묘한 색조. 와시 종이 화이트 (#F5F3F0), 먹물 그레이 (#3D3D3D), 대나무 그린 (#8B9A7E), 도자기 베이지 (#E8E3DB). 모든 색상은 채도 20% 이하로 제한하여 차분함 유지.

**Layout Paradigm**: 
수직 스크롤 중심의 단일 컬럼 레이아웃. 각 제품은 전체 뷰포트 높이를 차지하며, 스크롤 시 부드럽게 페이드 전환. 좌우 여백은 뷰포트의 15%를 유지하여 시선을 중앙에 집중.

**Signature Elements**:
- 종이 질감 오버레이 (미세한 노이즈 패턴, opacity 0.03)
- 붓글씨 스타일의 구분선 (불규칙한 두께, SVG path)
- 제품 이미지 주변의 넓은 여백 (최소 80px)

**Interaction Philosophy**: 
"무위(無爲)"의 인터랙션. 사용자가 의식하지 못할 정도로 자연스러운 전환. 클릭보다는 스크롤 중심. 호버 효과는 극도로 절제 (투명도 변화만, 0.7 → 1.0).

**Animation**: 
모든 애니메이션은 "호흡"을 모방. 페이드인은 3초에 걸쳐 천천히 진행 (ease-in-out). 스크롤 기반 애니메이션은 사용자 속도에 완벽히 동기화. 갑작스러운 움직임 금지.

**Typography System**: 
- Display: Noto Serif KR (300, 700) - 제품명, 주요 헤딩
- Body: Noto Sans KR (300, 400) - 본문
- Accent: Nanum Myeongjo (400) - 강조 문구
행간: 1.8 (동양 서체의 가독성 최적화)
자간: 0.02em (여유로운 호흡)
</text>

## 아이디어 3: Kinetic Luxury

**Design Movement**: 키네틱 아트 + 하이엔드 패션 브랜딩

**Core Principles**:
- 움직임이 곧 정체성 (모든 요소가 미묘하게 살아있음)
- 레이어드 깊이감 (5단계 이상의 Z-depth)
- 대담한 스케일 변화와 극적인 대비
- 럭셔리 브랜드의 배타성 표현

**Color Philosophy**: 
다크 럭셔리 팔레트. 미드나잇 네이비 (#0A0E27), 샴페인 골드 (#D4AF37), 플래티넘 실버 (#E5E4E2), 딥 버건디 (#4A0E0E). 그라데이션을 적극 활용하되 미묘하게 (5-10도 색상 변화).

**Layout Paradigm**: 
대각선 기반 동적 그리드. 제품 카드가 15도 기울어진 상태로 배치되며, 스크롤 시 회전하며 정렬됨. 풀블리드 이미지와 텍스트 오버레이의 극적 조합. 마우스 움직임에 반응하는 패럴랙스 레이어.

**Signature Elements**:
- 홀로그래픽 그라데이션 보더 (무지개빛 반사 효과, conic-gradient)
- 3D 변형 카드 (perspective: 1000px, rotateY on hover)
- 글래스모피즘 UI 요소 (backdrop-blur-xl, 반투명 배경)

**Interaction Philosophy**: 
"반응하는 럭셔리". 모든 인터랙션이 즉각적이고 유려함. 버튼은 호버 시 3D로 떠오름 (translateZ 10px). 커서를 따라다니는 커스텀 포인터 (40px 원형, mix-blend-mode: difference).

**Animation**: 
지속적인 미세 애니메이션. 배경 그라데이션이 15초 주기로 회전 (hue-rotate). 제품 이미지는 마우스 위치에 따라 3D 회전 (최대 ±15도). 페이지 전환 시 리퀴드 모션 효과 (GSAP 활용).

**Typography System**: 
- Display: Cormorant Garamond (600, 700) - 우아한 세리프, 제품명
- Body: Montserrat (300, 400, 600) - 모던 산세리프, 설명
- Accent: Cinzel (400) - 클래식 세리프, 가격/스펙
크기 대비: 96px (hero) / 32px (h2) / 18px (body) / 14px (caption)
Letter-spacing: Display -0.02em (타이트), Body 0.01em (여유)

## 선택된 디자인: Kinetic Luxury

**선택 이유**: 
"좆간지나게"라는 요구사항에 가장 부합하는 접근. 움직임, 깊이감, 대담한 스타일링이 프리미엄 제품 쇼케이스에 최적화되어 있으며, 현대적이면서도 럭셔리한 느낌을 극대화할 수 있음.

**핵심 구현 요소**:
1. 다크 럭셔리 컬러 팔레트 (미드나잇 네이비, 골드, 실버)
2. 3D 변형 및 글래스모피즘 UI
3. 마우스 인터랙션 기반 패럴랙스
4. 홀로그래픽 그라데이션 효과
5. Cormorant Garamond + Montserrat 타이포그래피 조합
